2. Digital Elevator Control System

Problem Statement

Elevator systems require reliable control logic to manage floor requests and ensure accurate stopping at each floor.

Approach

• Designed an 8-floor elevator control system using digital logic
• Implemented sequential control with TTL integrated circuits
• Developed a working prototype using breadboard before final assembly

Key Components

• 74LS148 Priority Encoder
• 74190 Up/Down Counter
• NE555 Timer
• LM339 Comparator

Outcome

• Successfully demonstrated sequential elevator movement
• Achieved accurate floor detection and stopping logic

Tech Stack

Digital Electronics, Sequential Logic Design, Hardware Prototyping

---

3. Vibration Monitoring System using ESP32

Problem Statement

Industrial machines require continuous monitoring to detect abnormal vibration patterns that may indicate faults or mechanical issues.

Approach

• Developed an IoT-based vibration monitoring system
• Used sensors connected to an ESP32 microcontroller
• Enabled real-time monitoring and data acquisition

Outcome

• Demonstrated integration of sensors and IoT communication
• Showed potential for predictive maintenance applications

Tech Stack

ESP32, Embedded C, Sensors, IoT System Integration
